# GitHub Foundations Quizzes

Test your knowledge with these practice questions!

## Quiz 1: Git Basics

**Question 1:** What command is used to initialize a new Git repository?

- A) `git start`
- B) `git init`
- C) `git create`
- D) `git new`

**Answer:** B) `git init`

**Explanation:** The `git init` command creates a new Git repository in the current directory.

---

**Question 2:** What does `git clone` do?

- A) Creates a new branch
- B) Copies a repository to your local machine
- C) Deletes a repository
- D) Merges two branches

**Answer:** B) Copies a repository to your local machine

**Explanation:** `git clone` creates a local copy of a remote repository.

---

## Quiz 2: GitHub Collaboration

**Question 3:** What is a Pull Request?

- A) A request to delete code
- B) A method to propose changes to a repository
- C) A way to download files
- D) A security feature

**Answer:** B) A method to propose changes to a repository

**Explanation:** Pull requests let you tell others about changes you've pushed to a branch in a repository.

---

**Question 4:** Which of the following is NOT a benefit of code review?

- A) Catches bugs early
- B) Improves code quality
- C) Slows down development intentionally
- D) Shares knowledge

**Answer:** C) Slows down development intentionally

**Explanation:** While code review takes time, its purpose is to improve quality, not to slow development.

---

## Quiz 3: Repository Management

**Question 5:** What is the main branch typically called in Git?

- A) `master` or `main`
- B) `primary`
- C) `root`
- D) `default`

**Answer:** A) `master` or `main`

**Explanation:** The default branch is traditionally called `master`, though many projects now use `main`.

---

**Question 6:** What does forking a repository do?

- A) Deletes the original repository
- B) Creates a personal copy of someone else's repository
- C) Renames the repository
- D) Archives the repository

**Answer:** B) Creates a personal copy of someone else's repository

**Explanation:** Forking creates a copy of a repository under your account where you can freely experiment.
